# 🎬 미디어 플레이어 프로젝트 (My Media Player)

> **C# Windows Forms 기반의 유튜브 모티브 미디어 재생 및 공유 플랫폼**

이 프로젝트는 유튜브(YouTube)를 모티브로 하여 개발된 데스크톱 미디어 플레이어 어플리케이션입니다. 기본적인 영상 및 음악 재생 기능뿐만 아니라, 회원가입, 프리미엄 멤버십 결제, 영상 업로드 및 구독 관리 등 실제 플랫폼과 유사한 사용자 경험을 제공하는 것을 목표로 합니다.

---

## ✨ 주요 기능 (Key Features)

이 어플리케이션은 다음과 같은 핵심 기능들을 포함하고 있습니다.

### 1️⃣ 미디어 재생 및 관리 (Media Playback)
* **영상 재생**: `VideoView`를 통해 선택한 영상을 재생하며, 전체 화면, 볼륨 조절 등 기본적인 제어 기능을 지원합니다.
* **음악 전용 모드**: `MusicPlay` 기능을 통해 영상 없이 오디오 파일(.mp3)만 별도로 리스트업하고 재생할 수 있는 뮤직 플레이어 모드를 제공합니다.
* **콘텐츠 업로드**: 사용자가 자신의 PC에 있는 영상 파일을 업로드하여 플랫폼에 등록할 수 있습니다.

### 2️⃣ 사용자 멤버십 및 결제 (Membership & Payment)
* **회원 시스템**: 회원가입(`NewMember`), 로그인(`Login`), 아이디/비밀번호 찾기(`IDfnd`, `PWfind`) 등 완벽한 계정 관리 기능을 구현했습니다.
* **프리미엄 구독**: 유료 멤버십 가입을 위한 결제 프로세스를 시뮬레이션합니다. 카드 정보 입력(`Cardplus`) 및 결제 처리(`Pay`) 로직이 포함되어 있습니다.
* **마이페이지**: 사용자의 닉네임, 등급(일반/프리미엄), 포인트, 구독 정보 등을 한눈에 확인하고 관리할 수 있습니다.

### 3️⃣ 관리자 모드 (Admin)
* **관리자 기능**: `ManagerForm`을 통해 전체 회원 관리나 콘텐츠 관리 등 일반 사용자가 접근할 수 없는 관리자 전용 기능을 수행합니다.

## 📂 소스 코드 구조 (Directory Structure)

프로젝트의 주요 소스 코드는 다음과 같이 구성되어 있습니다.

### 🏗️ 메인 및 공통 (Core)
* **`Program.cs`**: 어플리케이션의 시작점입니다.
* **`MainForm1.cs`**: **[메인 로비]** 영상 목록을 보여주고 각 기능으로 이동하는 메인 화면입니다.
* **`UserSession.cs`**: 로그인한 사용자의 정보를 전역적으로 관리하는 세션 클래스입니다.

### 🔐 계정 및 인증 (Auth)
* **`Login.cs` / `NewMember.cs`**: 로그인 및 회원가입 처리를 담당합니다.
* **`IDfnd.cs` / `PWfind.cs`**: 분실한 아이디와 비밀번호를 찾는 기능을 제공합니다.

### 📺 재생 및 콘텐츠 (Media)
* **`VideoView.cs`**: 영상 재생 화면 및 플레이어 로직이 구현되어 있습니다.
* **`MusicPlay.cs`**: 음악 재생 전용 인터페이스입니다.
* **`Upload.cs` / `Upload2.cs`**: 파일 업로드 및 메타데이터 입력 창입니다.

### 💳 결제 및 마이페이지 (Service)
* **`MyPage.cs`**: 내 정보 관리, 구독 현황 등을 표시합니다.
* **`Pay.cs` / `Cardplus.cs`**: 결제 수단 선택 및 카드 정보 입력 시뮬레이션입니다.

## ⚠️ 주의사항 및 실행 가이드 (Notes)

> 프로젝트 실행 및 데이터베이스 연동 시 유의사항입니다.

### 1. 🗄️ 데이터베이스 연결 (DB Connection)
이 프로젝트는 **MySQL** 데이터베이스를 사용합니다. `Login.cs` 등에서 DB 연결 문자열(Connection String)을 확인하고, 본인의 로컬 DB 환경(Port, ID, PW)에 맞게 수정해야 정상적으로 로그인 및 데이터 조회가 가능합니다.

### 2. 📂 미디어 파일 경로
영상이나 음악 재생 시 파일의 경로가 올바른지 확인해야 합니다. 업로드된 파일이 저장되는 기본 폴더가 존재하지 않을 경우 오류가 발생할 수 있으므로, 실행 전 해당 경로를 확인해 주세요.

### 3. 🛡️ 관리자 권한
`ManagerForm` 기능은 특정 관리자 계정으로 로그인했을 때만 활성화되거나 접근 가능할 수 있습니다.
