# 🎬 미디어 플레이어 프로젝트 (My Media Player)


이 프로젝트는 C#을 활용하여 개발한 유튜브를 모티브로 한 미디어 플레이어 기본적인 영상 재생 , 미리보기 기능 , 유료 계정 가입, 뮤직 재생 , 마이페이지 , 구독/댓글 기능 등 다양한 편의 기능 제공

---

## ✨ 주요 기능

이 미디어 플레이어는 다음과 같은 핵심 기능들을 포함하고 있습니다

### 1. 미디어 재생 및 제어
* **기본 영상 재생**: 로컬 및 (스트리밍 URL?) 영상의 재생, 일시정지, 정지, 볼륨 조절, 전체 화면 기능을 지원
* **뮤직 재생**: 영상과 별도로 오디오 파일(.mp3 등)을 관리하고 재생할 수 있는 '뮤직' 탭 또는 모드를 제공
* **미리보기**: 영상 목록의 썸네일에 마우스를 올렸을 때 (또는 재생 바를 탐색할 때) 해당 부분의 이미지를 미리보기


### 2. 사용자 계정 및 인증
* **회원가입 및 로그인**: 사용자가 계정을 생성하고 로그인/로그아웃 기능 
* **유료 계정 가입**: '프리미엄' 계정으로 업그레이드할 수 있는 결제 (또는 등급 변경) 기능을 구현 (예: 광고 제거, 전용 콘텐츠 접근 등)
* **마이페이지**: 사용자가 자신의 프로필 정보, (업로드한 영상), 구독 목록, 결제 상태 등을 한눈에 확인하고 관리


### 3. 커뮤니티 및 소셜
* **구독**: 다른 사용자(채널)를 구독하거나 구독을 취소할 수 있습니다.
* **댓글**: 각 영상 하단에 사용자가 댓글을 작성, 조회, (수정/삭제)할 수 있는 기능을 제공합니다.


## 🛠️ 사용 기술 (Tech Stack)

이 프로젝트는 다음과 같은 기술 스택을 기반으로 개발

* **언어 (Language):**
    * `C#`
* **UI 프레임워크 (Framework):**
    * `Windows Forms (WinForms)`: .NET 환경의 데스크톱 애플리케이션 UI를 구현
* **데이터베이스 (Database):**
    * `MySQL`: 사용자 정보, 유료 계정 상태, 구독/댓글 내역, 미디어 메타데이터 등을 저장하고 관리
* **개발 도구 (Tools):**
    * `(예: Visual Studio 2022)`
    * `(예: MySQL Workbench)`
    * `Git & GitHub`: 형상 관리 및 팀 협업을 위해 사용
