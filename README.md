# 🎬 미디어 플레이어 프로젝트 (My Media Player)


이 프로젝트는 C#을 활용하여 개발한 유튜브를 모티브로 한 미디어 플레이어 기본적인 영상 재생 , 미리보기 기능 , 유료 계정 가입, 뮤직 재생 , 마이페이지 , 구독/댓글 기능 등 다양한 편의 기능 제공

---

## ✨ 주요 기능

이 미디어 플레이어는 다음과 같은 핵심 기능들을 포함하고 있습니다.

### 1. 미디어 재생 및 제어
* **기본 영상 재생**: 로컬 및 (스트리밍 URL?) 영상의 재생, 일시정지, 정지, 볼륨 조절, 전체 화면 기능을 지원합니다.
* **뮤직 재생**: 영상과 별도로 오디오 파일(.mp3 등)을 관리하고 재생할 수 있는 '뮤직' 탭 또는 모드를 제공합니다.
* **미리보기**: 영상 목록의 썸네일에 마우스를 올렸을 때 (또는 재생 바를 탐색할 때) 해당 부분의 이미지를 미리 볼 수 있습니다.

![기본 재생 화면](images-placeholder/screenshot_1.png)

### 2. 재생 목록 (Playlist) 관리
- **목록 조회**: '재생목록' 탭에서 현재 추가된 미디어 리스트를 한눈에 볼 수 있습니다.
- **파일 추가**: 재생 목록에 새로운 파일을 추가합니다.
- **파일 삭제**: 목록에서 선택한 파일을 삭제하거나, 전체 목록을 한 번에 비울 수 있습니다.

![재생 목록 관리](images-placeholder/screenshot_3.png)
![재생 목록 메뉴](images-placeholder/screenshot_4.png)

### 3. 환경설정
- **시작 설정**: '프로그램 시작 시 마지막 파일 실행' 옵션을 통해 사용 편의성을 높였습니다.
- **스킨 변경**: '기본' 스킨을 포함하여 총 4가지의 테마(스킨)를 제공합니다. 사용자는 원하는 스킨을 선택하여 플레이어의 외형을 즉시 변경할 수 있습니다.

![환경설정](images-placeholder/screenshot_6.png)
![스킨 변경 적용 예시](images-placeholder/screenshot_9.png)
